# Pinhalense auxílio de pesquisa e automatização – Estudo de Caso

## RN01 — Privacidade dos Clientes

 O Sistema irá censurar os dados pessoais do cliente durante o processo de recebimento de Croquis e Layouts.

## RN02 — Prevenção de equívocos

 A IA irá realizar os cálculos e com base nas regras da Pinhalense para evitar os erros de cálculos.
 
## RN03 — Realização de Backups

  O Banco de dados da IA irá realizar e backups de todos os dados e manterá criptografados.
